# MemoryShogame

## 13.07.2021

Simple juego de memoria con cartas usando Javascript + HTML  + CSS :nerd_face:	

El proyecto se divide en:
- css
- imgs
- Javascript - en js/main.js se encuentra toda la lógica del juego.
- index.html - Estructura de la página.
